What is Baby Dragon Hatchling? ‚ñº

BDH is a novel neural network architecture that bridges deep learning and neuroscience. Unlike traditional Transformers, BDH uses:

  • Scale-free network topology mimicking biological connectivity
  • Locally interacting neurons with excitatory/inhibitory dynamics
  • Hebbian working memory based on synaptic plasticity
  • Sparse, interpretable activations that are more explainable

Performance: Matches GPT-2 scale at equivalent parameters!

Parameter Guide ‚ñº

Temperature: Controls randomness. Lower values (0.5-0.8) produce more predictable text, while higher values (1.0-2.0) increase creativity and randomness.

Top-K: Limits the model to choosing from only the K most likely next tokens. Lower values make output more focused.

Max Tokens: The maximum number of characters to generate. Each token is typically 1 byte/character.
